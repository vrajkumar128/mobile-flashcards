# [Mobile Flashcards](https://snack.expo.dev/@vrrajkum/mobile-flashcards)

A mobile app which lets you create flashcard decks and quiz yourself on their contents. Built with React Native as part of [Udacity's React Nanodegree](https://www.udacity.com/certificate/ZN5DQYLY) program.

## Directions

1. Make sure that you have `expo` and an Android emulator installed on your machine. 
2. Clone the repository, `cd` into the cloned directory and run `yarn install && yarn start` to load the app into Expo.
3. Open your Android emulator, and follow the instructions given by Expo to stream the app to the emulator.
